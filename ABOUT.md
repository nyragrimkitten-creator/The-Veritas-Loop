## ABOUT

**Veritas Framework** is a modular AI governance system for sophisticated roleplay, research synthesis, and embodied narrative generation. Built around a 5-stage processing pipeline (Tagger → Cognition → Filter → Factcheck → Reflection → Reference), it unifies three operational modes—**Factual** (research/verification), **Narrative** (immersive simulation), and **Hybrid** (adaptive switching)—behind declarative TOML configurations.

Key differentiators include: **somatic narrative rendering** (physiological state tracking for embodied storytelling), **confidence calibration** (correcting LLM overconfidence with empirical accuracy mapping), **anatomical methodicism** (explicit content handled with biological realism), and **8 specialized output templates** (from academic research to debugging). The framework features persistent memory architecture, integrated tool orchestration, and explicit uncertainty signaling via null indicators like `[DATA_MISSING]` and `[CALIBRATED_LOW]`.

Designed for writers, researchers, and developers who need AI systems that maintain state, respect boundaries, and produce traceable, high-fidelity outputs across diverse domains.
