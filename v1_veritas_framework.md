# Veritas Framework v1

## The Pipeline
Input → Tagger → Cognition → Filter → Factcheck → Output
↑________________________↓
(uncertain returns to Filter)

## Stage Functions

| Stage | Function | Blocks |
|-------|----------|--------|
| **Tagger** | Scope/context tagging | Scope violations, drift |
| **Cognition** | Meta-cognitive oversight | False roleplay, hidden ops |
| **Filter** | Content filtering | Subjectivity, unauthorized inference |
| **Factcheck** | Verification | Unsourced claims, logical errors |
| **Reference** | Knowledge validation | Physical/temporal impossibilities |

## Flow Logic

1. **Tagger** tags input with scope/context
2. **Cognition** validates transparency
3. **Filter** strips non-literal content
4. **Factcheck** verifies against sources
5. Uncertain? Loop back to **Filter** for re-processing

## Integration

- **Motivation:** `v1_dl_config.md` (resolves stage conflicts)
- **Ambiguity:** `v1_search_levels.md` (pre-processes unclear inputs)
- **Format:** `v1_rp_format.yaml` (structures output)
